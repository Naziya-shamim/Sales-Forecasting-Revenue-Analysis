# Sales Forecasting & Revenue Trend Analysis

## 📌 Project Objective
To forecast future sales and analyze revenue trends using historical data, and build an interactive dashboard in Power BI for sales, finance, and management teams.

## 📊 Business Problem
Accurate sales forecasting helps companies plan inventory, set budgets, allocate marketing spend, and make strategic decisions. This dashboard provides a clear view of past performance and future predictions.

## 📁 Dataset Description
- **Source:** Generated realistic dataset (500 orders)
- **Columns:** Order ID, Order Date, Revenue, Profit, Forecasted Revenue, Product Category, Region, Target Achievement %, etc.

## 🛠️ Tools Used
- Power BI Desktop (DAX, Power Query)
- GitHub (Portfolio hosting)

## 📈 Key Metrics (KPIs)
| KPI | Value |
|-----|-------|
| Total Revenue | ₹2.09 billion |
| Revenue Growth % | 19.54% |
| Forecast Accuracy | 95.10% |
| Profit Margin | 30.21% |
| Avg Target Achievement | 87.88% |

## 📊 Dashboard Features
- **KPI Cards** – Top-level metrics
- **Revenue Trend** – Line chart by month/quarter
- **Forecasted vs Actual** – Clustered column chart
- **Product Category Analysis** – Bar chart
- **Regional Revenue** – Bar chart
- **Seasonal Revenue** – Table analysis
- **Monthly Variance** – Table analysis
- **Slicers** – Region, Product Category, Year
- **Insights & Recommendations** page

## 🔍 Key Insights
- Electronics contributes 52% of total revenue with 30.3% margin.
- All regions exceeded sales targets (South highest at 114.8%).
- Winter (Q4) is peak season; Spring lowest.
- Forecast accuracy is excellent (95.1%), but monthly variances exist (Jan +94%, May -24%).

## ✅ Recommendations
- Adjust monthly forecast coefficients for Jan/Dec and May/Feb.
- Increase marketing spend on Electronics.
- Boost East region revenue with targeted campaigns.
- Set realistic 2026 targets (normalize difficulty across regions).
